Making of an SDK and a prototype LDES

LDES2Service

LDES2Service is a software development kit (SDK) that makes it easier for developers to use datasets published as Linked Data Event Streams (LDES). The SDK allows developers to seamlessly replicate any LDES to a datastore of their choice, while also staying synchronized with new updates.

Linked Connections Server

The Linked Connections Server is now extended to be able to fetch GTFS and GTFS-RT data and to redistribute them as an LDES while being compliant with Linked Connections.
